Analytics Module

Description

StreetHawk's Analytics Module is core component of StreetHawk SDK. The module reports various vital raw data to StreetHawk servers which is further processed and used for various analytics for your application. The module also provides API for tagging the install, identifying the users and tracking various important events inside your application. All other components of StreetHawk depends on Analytics component for it's functioning.

Accessing the Analytics Module

To access this module from JavaScript, you would do the following:

var shAnalytics = require("com.streethawk.Analytics");

The shAnalytics variable is a reference to the Module object.

analytics.function

  • Initialize streethawk SDK

    shAnalytics.streethawkinit();

  • Open url handler

    function openurlCallback(e) { alert("Callback for open url: " + e.url); } shAnalytics.shDeeplinking({openurl: openurlCallback}); //must use key "openurl"

  • Tag cuid

    var tagSuccess = shAnalytics.tagCuid();

  • Tag numeric

    var tagSuccess = shAnalytics.tagNumeric("Key_Numeric", 100);

  • Tag string

    var tagSuccess = shAnalytics.tagString("Key_String", "abc");

  • Tag date

    var tagSuccess = shAnalytics.tagDatetime("Key_Date", "2015-12-08 10:12:16");

  • Increment tag

    var tagSuccess = shAnalytics.incrementTag("Key_Numeric");

  • Remove tag

    var tagSuccess = shAnalytics.removeTag("Key_Remove");

  • Send simple feedback

    shAnalytics.shSendSimpleFeedback("feedback title", "feedback message");

  • Log enter page

    shAnalytics.notifyViewEnter("page_name");

  • Log exit page

    shAnalytics.notifyViewExit("page_name");

analytics.property

  • Set or get app_key

    shAnalytics.appKey = ; var app_key = shAnalytics.appKey;

  • Set or get console debug mode

    shAnalytics.enableLogs = ; var enableLogs = shAnalytics.enableLogs;

  • Set or get itunes id

    shAnalytics.shiTunesId = ; var iTunesId = shAnalytics.shiTunesId;

  • Set or get advertisement id

    shAnalytics.advertisementId = ; var advertisementId = shAnalytics.advertisementId;

  • Get SDK version

    var sdkVersion = shAnalytics.getSHLibraryVersion;

  • Get Install id string

    var installId = shAnalytics.getInstallId;

  • Get current time

    var currentTime = shAnalytics.getCurrentFormattedDateTime;

Usage

  • Tag sh_phone

    tagSuccess = shAnalytics.tagString("sh_phone", "+123456");

  • Tag sh_email

    tagSuccess = shAnalytics.tagString("sh_email", "abc@gmail.com");
